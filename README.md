# SightWalk (iOS)

iOS project for the SightWalk apps.
Uses Cocoapods for dependencies.

## Current dependencies
- Alamofire
- SwiftyJSON
- Locksmith
- JLToast
- GoogleMaps
- SwiftCop


## Instructions
Clone project and run ```pod install```.
Use the xcworkspace to work on the project.
